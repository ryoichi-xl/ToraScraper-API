path.join is the safest way to build paths:
e.g., path.join('images', 'cat.png') goes to images/cat.png

path.resolve builds absolute path:
path.resolve('images', 'cat.png') goes to C:\Users\User\project\images\cat.png

path.dirname() gets the folder of a file
path.dirname('/a/b/c.text) gives /a/build

path.basename() gets filename only
path.basename('/a/b/c.txt') gives c.txt

path.extname() gets file extension
path.extname('image.png') gives .png

# FS MODULE

Reading files (async version):
import fs from 'fs/promises'

const data = await fs.readFile('file.txt', 'utf-8')

Writing Files:
await fs.writeFile('file.txt', 'hello')

Create Folder:
await fs.mkdir('images')

Check if file exists:
await fs.access('file.txt')

Delete file
await fs.unlink('file.txt')

Rename file
await fs.rename('old.txt', 'new.txt')
